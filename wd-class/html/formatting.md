# Formatting Tags
## &lt;b> and &lt;strong>
``` html
<!DOCTYPE html>
<html>
    <head>
        <title>Formatting Tag</title>
    </head>
    <body>
        <p><b>This is bold text</b></p>
        <p><strong>This is strong text</strong></p>
    </body>
</html>
```
**Output:**
<!DOCTYPE  html>
<html>
    <body>
        <p><b>This is bold text</b></p>
        <p><strong>This is strong text</strong></p>
    </body>
</html>

## &lt;del> and &lt;ins>
```html
<!DOCTYPE html>
<html>
    <head>
        <title>Formatting Tags</title>
    </head>
    <body>
       <p>I like <del>dogs</del> <ins>cats</ins></p>
    </body>
</html>
```
**Output**

<!DOCTYPE html>
<html>
    <body>
       <p>I like <del>dogs</del> <ins>cats</ins></p>
    </body>
</html>
